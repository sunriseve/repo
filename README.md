# 🌌 Sunrise SkyStream Repository

A SkyStream plugin repository providing Movies, TV Shows, Bollywood, Hollywood & Cartoons streaming.

## 📦 Plugins Included

| Plugin | Description | Languages |
|--------|-------------|-----------|
| **AllMovieLand** | Movies, TV Shows, Bollywood, Hollywood & Cartoons | en, hi |

---

## 🚀 Getting Started

### 1. Installation
To install SkyStream on your device:
- **Download:** Navigate to the [SkyStream releases page](https://github.com/akashdh11/skystream/releases/) and download the latest release.
- **Install:** Open the downloaded file and follow your system's installation prompts.
- **Launch:** Once installed, open the **SkyStream** app.

---

## 🛠 Setting Up This Repository

### Add Repository URL
```
https://raw.githubusercontent.com/sunriseve/repo/master/repo.json
```

### Installation Steps
1. Open SkyStream app
2. Navigate to **Settings** → **Manage Extensions**
3. Click **Add Repository**
4. Paste the Repository URL above
5. Tap **Add**
6. Wait for plugins to populate, then **download** AllMovieLand

---

## 📺 Using the Plugin

1. Return to the **Home Screen**
2. Tap the **Provider** button (bottom right FAB)
3. Select **AllMovieLand**
4. Browse Movies, TV Shows, Bollywood, Hollywood, and Cartoons

### Supported Categories
- Movies
- Bollywood Movies
- Hollywood Movies
- TV Shows
- Cartoons
